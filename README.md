## Highlights

**[Galactis](https://github.com/estevansantos-ops/Galactis-MIPS)**
A Bomberman-style game written entirely in MIPS Assembly, rendered on a bitmap display via Memory-Mapped I/O in the MARS 4.5 simulator.

**[Perceptual Hash — MIPS](https://github.com/estevansantos-ops/perceptual-hash-mips)**
Image authentication system that detects visual similarity using a hybrid Python + MIPS Assembly pipeline: Python handles DCT preprocessing, Assembly computes the 64-bit pHash and Hamming distance at the register level.

**[Smart Assessment](https://github.com/estevansantos-ops/sistema-avaliacao-inteligente)**
Fullstack academic platform (Next.js, MongoDB, Docker) that uses LLMs to generate exam questions and OCR to auto-grade answer sheets, with real-time analytics dashboards and LaTeX PDF export.
